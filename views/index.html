Time Management System

A web application for planning and organizing daily time using schedule-based time blocks. The system focuses on time allocation rather than task tracking.

Team - SE-2427

  • Yesset Meiirzhan Miras Nurtore

Planned Features

  • Create daily schedules using time blocks
  • Assign activity types to time blocks
  • View daily and weekly schedules
  • Edit and rearrange time blocks
  • Visualize time distribution
